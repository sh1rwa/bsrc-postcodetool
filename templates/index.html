Bristol Postcode to Ward Lookup

▼ How to use this tool
  • Enter one or more postcodes into the box below.
  • Separate postcodes using commas or new lines.
  • Click Lookup to find the corresponding ward.
  • Invalid inputs will be highlighted, and suggestions will be provided where possible.
  • You can enter duplicate postcodes if needed.

Additional features

  • Click Show/Hide Chart to see a bar chart breakdown of postcodes by ward.
  • Use the Filter button to select specific wards using tickboxes, as well as the option to show/hide invalid/not found postcodes.
  • The table, chart, and totals will update automatically based on your selected filters.
  • Click Reset to default in the filter menu to restore all wards.
  • Click Save CSV to export your results into a CSV file that can be used in Excel.
  • Click Save Chart to download the chart with per-ward and total postcode counts.

Example:

BS1 1AD
BS1 1BT, BS1 1BU

Note:

'Invalid' or 'Not Found' postcodes will be highlighted, but will NOT be included in the charts or CSV.
